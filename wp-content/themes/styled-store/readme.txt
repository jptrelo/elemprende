== Description ==

StyledStore is the perfect theme designed for your WooCommerce shop.Styled Store is designed to provide clarity, maximum content exposure and ease in navigation.

== Limitation ==

Footer menu is limited by single depth and have seperate menu location called Footer Menu.
Different Menu Location for Mobile Version, So please seperate Menu for Mobile. 

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.


Styledstore includes support for Infinite Scroll in Jetpack.

== Information ==

Footer menu depth is limited by single depth
Search Bar is attached to primary menu location
