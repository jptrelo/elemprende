== Description ==
*The only plugin in Wordpress Directory which does that!*
**Please, review this plugin =>**

This plugin creates a TAB on the Woocommerce My Account page called "Waiting for Review", making it easy to review all products purchased. 

1. CSS 100% compatible with yout theme.
1. Display the pure form from the Woocommerce Review Tab. 

Shortcode:
* [awaitingreviews] : Show where you want the product review form, for those who has not yet been evaluated. 

*Fell free to help me improve it, made with love.*

== Installation ==

Pretty simple to install the plugin:
1. Upload `woo-awaiting-reviews.zip` to Wordpress backend.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Voilà - Ready to use

== Frequently Asked Questions ==
= Where can I configure this plugin? =
No need that, this is simple as eating cake. Just activate go to Wocommerce My Account, and you gonna find a "Awaiting Reviews" there.

= Why there's only one product shown? =
This is because it works at one product review at time. It's great because pushes the customer to review all!
